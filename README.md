# ☕ 百分之一咖啡店员工排班系统

一个基于纯前端技术的咖啡连锁店员工排班管理系统。

## 🌟 在线访问

🔗 **[点击这里访问系统](https://你的用户名.github.io/coffee-shop-scheduler/)**

## 📋 功能特性

- ✅ **排班日历** - 可视化周视图排班管理
- ✅ **员工管理** - 15位员工信息管理
- ✅ **班次设置** - 早中晚班次自定义
- ✅ **分店管理** - 支持9个分店独立排班
- ✅ **数据导出** - 排班表图片/Excel导出
- ✅ **数据备份** - 本地数据导入/导出

## 🚀 技术栈

- **前端**: HTML5 + CSS3 + 原生JavaScript
- **存储**: localStorage（浏览器本地存储）
- **部署**: GitHub Pages

## 💻 本地运行

### 方法1：直接打开
双击 `index.html` 文件即可在浏览器中运行

### 方法2：使用本地服务器
```bash
python server.py
```
然后访问 http://localhost:8080

## 📱 移动端支持

系统采用响应式设计，完美支持手机、平板访问。

## 📦 数据说明

- 数据存储在浏览器 localStorage 中
- 不同设备数据独立，不自动同步
- 建议定期使用"导出数据"功能备份
- 可通过"导入数据"功能在设备间迁移数据
