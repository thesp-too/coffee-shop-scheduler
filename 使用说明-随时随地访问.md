# 📱 手机随时随地访问排班系统

## 🚀 方案一：ngrok 内网穿透（推荐，最简单）

### 优点
- ✅ 5分钟快速部署
- ✅ 免费版本够用
- ✅ 自动生成HTTPS链接
- ✅ 无需购买服务器

### 缺点
- ⚠️ 免费版每次启动链接会变
- ⚠️ 需要电脑保持开机运行

### 操作步骤

#### 1. 下载 ngrok
访问 https://ngrok.com/download 下载Windows版本

#### 2. 安装和注册
- 注册免费账号：https://dashboard.ngrok.com/signup
- 登录后获取你的 authtoken
- 运行命令配置：`ngrok config add-authtoken <你的token>`

#### 3. 启动服务
打开PowerShell，执行：
```powershell
# 进入项目目录
cd c:\05.Practice\coffee-shop-scheduler

# 启动本地服务器（新开一个终端）
python server.py

# 启动ngrok（另开一个终端）
ngrok http 8080
```

#### 4. 获取公网地址
ngrok启动后会显示：
```
Forwarding  https://xxxx-xx-xxx-xxx-xxx.ngrok-free.app -> http://localhost:8080
```

#### 5. 手机访问
用手机浏览器打开上面的 https://xxxx.ngrok-free.app 链接即可！

## 🌐 方案二：GitHub Pages（完全免费，永久在线）

### 优点
- ✅ 完全免费
- ✅ 24小时在线
- ✅ 固定网址
- ✅ 电脑不需要开机

### 缺点
- ⚠️ 需要推送代码到GitHub
- ⚠️ 每次更新需要重新部署

### 操作步骤

#### 1. 创建GitHub仓库
- 登录 https://github.com
- 创建新仓库，命名如 `coffee-shop-scheduler`
- 设置为Public

#### 2. 上传代码
在项目目录打开PowerShell：
```powershell
# 初始化git仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 关联远程仓库（替换为你的GitHub用户名）
git remote add origin https://github.com/你的用户名/coffee-shop-scheduler.git

# 推送到GitHub
git push -u origin main
```

#### 3. 启用GitHub Pages
- 进入仓库的 Settings
- 找到 Pages 选项
- Source 选择 `main` 分支
- 保存

#### 4. 访问网站
几分钟后，访问：`https://你的用户名.github.io/coffee-shop-scheduler/`

## 🔧 方案三：Cloudflare Tunnel（免费+稳定）

### 优点
- ✅ 完全免费
- ✅ 固定域名
- ✅ 不限时长
- ✅ 更稳定

### 操作步骤

#### 1. 安装 cloudflared
下载：https://github.com/cloudflare/cloudflared/releases

#### 2. 登录认证
```powershell
cloudflared tunnel login
```

#### 3. 创建隧道
```powershell
cloudflared tunnel create coffee-shop
```

#### 4. 启动隧道
```powershell
# 先启动本地服务器
python server.py

# 再启动cloudflare隧道
cloudflared tunnel --url http://localhost:8080
```

## ⚠️ 重要提示

### 数据安全
- 这个系统使用 localStorage 存储数据
- 数据保存在浏览器本地
- 更换浏览器或清除缓存会丢失数据
- 建议定期使用系统内置的"导出数据"功能备份

### 多设备使用
- 不同设备的数据不会自动同步
- 如需同步，需要手动导入/导出数据
- 或考虑添加云端数据库功能（需要改造）
