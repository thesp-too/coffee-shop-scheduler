# ☕ 百分之一咖啡店员工排班系统

一个专为咖啡连锁店设计的员工排班管理系统，采用纯前端技术，支持多分店管理，完美适配移动端。

## 🌟 在线访问

🔗 **[点击访问排班系统](https://thesp-too.github.io/coffee-shop-scheduler/)**

支持手机、平板、电脑全平台访问，随时随地管理排班！

---

## ✨ 核心功能

### 📅 排班日历
- **周视图显示** - 一目了然查看整周排班
- **可视化操作** - 点击日期快速分配班次
- **拖拽排班** - 支持拖拽员工到班次（电脑端）
- **快速导航** - 上一周/下一周/今天快速切换
- **图片导出** - 一键生成排班表图片分享

### 👥 员工管理
- **17位员工** - 预设员工信息，可随时添加/编辑/删除
- **颜色标识** - 每位员工独特颜色，日历显示更清晰
- **职位管理** - 支持店长、咖啡师等多种职位
- **联系信息** - 记录电话、邮箱等联系方式

### ⏰ 班次设置
- **3种默认班次** - 早班、中班、全班
- **灵活调整** - 时间可自定义，适应不同门店需求
- **颜色区分** - 不同班次不同颜色，一眼识别

### 🏪 分店管理
- **9家分店** - 联庄、余杭、星光、开元、山泽里、采荷、泰隆、浣纱、工厂
- **独立排班** - 每个分店独立管理排班
- **快速切换** - 顶部下拉菜单快速切换分店

### 📱 移动端优化
- **响应式设计** - 完美适配手机竖屏
- **汉堡菜单** - 手机端点击 ☰ 按钮访问所有功能
- **简洁界面** - 移动端隐藏次要信息，专注核心功能
- **触摸友好** - 按钮大小适合手指操作

### 💾 数据管理
- **本地存储** - 数据保存在浏览器 localStorage
- **自动备份** - 每5秒自动保存备份
- **导入导出** - 支持 JSON 格式数据导入导出
- **一键重置** - 重置数据恢复到默认状态

## 📱 使用指南

### 电脑端
1. 访问网址 https://thesp-too.github.io/coffee-shop-scheduler/
2. 顶部导航栏切换不同功能页面
3. 点击日历格子分配班次
4. 右侧可查看员工列表（已隐藏，可在代码中启用）

### 手机端
1. 访问网址（建议添加到主屏幕）
2. 点击右上角 **☰** 按钮打开菜单
3. 选择功能：排班日历、员工管理、班次设置、分店管理
4. 竖屏浏览，操作更便捷

### 常用操作
- **添加排班**：点击日历日期 → 选择员工和班次 → 保存
- **修改排班**：点击已有班次 → 编辑或删除
- **切换分店**：顶部下拉菜单选择分店
- **导出排班表**：点击"生成排班表"按钮
- **重置数据**：点击"重置数据"按钮恢复默认

## 📊 数据说明

### 数据存储位置
- **浏览器 localStorage** - 所有数据存储在本地浏览器
- **不同设备独立** - 手机和电脑数据不互通
- **按域名隔离** - file:// 和 https:// 访问数据独立

### 数据同步
系统**不支持云端同步**，如需在多设备间同步：
1. 在设备A点击"导出数据"下载 JSON 文件
2. 在设备B点击"导入数据"上传 JSON 文件

### 数据备份
- **自动备份**：每5秒自动保存到 localStorage
- **手动备份**：使用"导出数据"功能下载 JSON 文件
- **建议**：重要数据定期导出保存

---

## ⚠️ 注意事项

1. **浏览器兼容性**
   - 推荐使用 Chrome、Edge、Safari、Firefox 最新版
   - 需启用 JavaScript 和 localStorage

2. **数据安全**
   - 数据仅存储在浏览器本地，清除浏览器数据会丢失
   - 重要数据请及时导出备份
   - 不涉及服务器，数据完全私密

3. **移动端使用**
   - 建议添加到主屏幕，像App一样使用
   - 首次加载可能需要下载资源，请耐心等待
   - 离线后可继续使用（数据已缓存）

4. **重置数据**
   - 重置数据操作不可逆
   - 重置前请先导出数据备份
